# EarTrumpet – Smart Audio Volume Management & Productivity Tool

**EarTrumpet** is a powerful, flexible audio management solution designed to help individuals and teams control application volumes, track audio sources, and achieve sound clarity.  
With **EarTrumpet**, you can transform chaotic system volume into structured per-app control.

## Key Features of EarTrumpet

- **Per-App Volume** – **EarTrumpet** allows independent volume for each application.  
- **Output Device Routing** – **EarTrumpet** sends each app to different speakers or headphones.  
- **System Tray Integration** – Access **EarTrumpet** instantly from the notification area.  
- **Keyboard Shortcuts** – Control **EarTrumpet** without mouse interaction.  
- **Peak Metering** – **EarTrumpet** shows real-time audio levels for each app.  
- **Dark Mode** – Use **EarTrumpet** with system-wide dark theme support.  
- **Export Settings** – Export your **EarTrumpet** config to JSON or XML.  
- **Cross-Platform** – Use **EarTrumpet** on Linux, Android, iOS, and web.  
- **Push Notifications** – Get alerts from **EarTrumpet** when audio devices change.  
- **Preset Manager** – Save volume layouts inside **EarTrumpet** for reuse.  
- **Command Line Interface** – Control **EarTrumpet** from scripts and automation.  
- **Low Resource Mode** – Run **EarTrumpet** with minimal CPU usage.  
- **Hotkey Control** – Mute or adjust any app using **EarTrumpet** shortcuts.  
- **Startup Behavior** – Launch **EarTrumpet** automatically on login.  
- **Search & Filter** – Find any audio source instantly using **EarTrumpet** search.
